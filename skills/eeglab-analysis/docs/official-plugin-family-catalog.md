# Official EEGLAB Plugin Family Catalog

This catalog indexes the broader plugin families surfaced by the official EEGLAB plugins page. It is discovery-only: it helps users find official plugin families and decide whether to install or route work, but it does not promise executable support by itself.

Use `eeglab_plugin_check` and `docs/official-plugin-map.md` for support levels and gate decisions.
