# Source and Connectivity Advanced Figure Gallery

This default module keeps the source-space and connectivity review families
visible and aligned to the official plugin tutorials.

## Default Status

- Default enabled: yes
- Python module: `scripts/advanced_figures/source_gallery.py`
- Markdown guide: `scripts/advanced_figures/source.md`
- Default MCP route: `eeglab_source_localization / eeglab_plot_connectivity`

## Figure Families

| Figure family | Status | Official anchor | Official URL | MCP tool | Official scope |
| --- | --- | --- | --- | --- | --- |
| Source / dipole review | required | `EEGLAB-DIPFIT-001` | `https://eeglab.org/plugins/dipfit/` | `eeglab_source_localization` | Model fit quality and residual variance review. |
| ROI connectivity matrix | conditional | `EEGLAB-ROICONNECT-001` | `https://eeglab.org/plugins/roiconnect/` | `eeglab_plot_connectivity` | ROI-level functional connectivity summary. |
| ROI 2-D / 3-D review | guidance_only | `EEGLAB-ROICONNECT-001` | `https://eeglab.org/plugins/roiconnect/` | `ROIconnect` | Source-space review aid only. |
| SIFT time-frequency grids | guidance_only | `EEGLAB-SIFT-001` | `https://eeglab.org/plugins/SIFT/` | `SIFT` | Model-based connectivity summary only. |
