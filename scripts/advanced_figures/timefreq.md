# Time-Frequency Advanced Figure Gallery

This default module keeps the richer ERSP / ITC figure families visible and
aligned to the official time-frequency tutorials.

## Default Status

- Default enabled: yes
- Python module: `scripts/advanced_figures/timefreq_gallery.py`
- Markdown guide: `scripts/advanced_figures/timefreq.md`
- Default MCP route: `eeglab_plot_timefreq`

## Figure Families

| Figure family | Status | Official anchor | Official URL | MCP tool | Official scope |
| --- | --- | --- | --- | --- | --- |
| ERSP / ITC heatmap | required | `EEGLAB-TF-001` | `https://eeglab.org/tutorials/08_Plot_data/Time-Frequency_decomposition.html` | `eeglab_plot_timefreq` | Event-locked power and phase dynamics. |
| Time-frequency curves | conditional | `EEGLAB-TF-001` | `https://eeglab.org/tutorials/08_Plot_data/Time-Frequency_decomposition.html` | `eeglab_plot_timefreq` | Frequency-specific time courses. |
| tftopo summary | guidance_only | `EEGLAB-TF-001` | `https://eeglab.org/tutorials/08_Plot_data/EEG_scalp_measures.html` | `tftopo` | Multi-electrode summary, static only. |
| metaplottopo summary | guidance_only | `EEGLAB-TF-001` | `https://eeglab.org/tutorials/08_Plot_data/EEG_scalp_measures.html` | `metaplottopo` | Channel-grid summary, static only. |
