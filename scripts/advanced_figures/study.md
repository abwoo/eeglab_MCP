# STUDY Advanced Figure Gallery

This default module keeps the group-level figure families visible and aligned
to the official STUDY and group-statistics tutorials.

## Default Status

- Default enabled: yes
- Python module: `scripts/advanced_figures/study_gallery.py`
- Markdown guide: `scripts/advanced_figures/study.md`
- Default MCP route: `eeglab_plot_erp / eeglab_plot_psd / eeglab_plot_timefreq / eeglab_study_statistics`
