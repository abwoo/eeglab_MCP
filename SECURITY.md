# Security Policy

This repository is a local-first EEG / EEGLAB research workflow toolkit.

## Reporting

Please report security issues through GitHub Security Advisories or a private
repository advisory.

Include:

- affected file(s) or workflow(s)
- exact reproduction steps
- expected versus observed behavior
- any safe mitigation or rollback guidance

Do not include real EEG data, personal data, or other sensitive research
materials in security reports.

## Scope

Security reports should focus on:

- code execution or sandbox escape issues
- unsafe workflow or token handling
- dependency or supply-chain problems
- data handling or path traversal bugs in the MCP server and scripts

Clinical or diagnostic claims are out of scope for this repository.
